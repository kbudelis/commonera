# Cosmic Calendar

A mobile-first Hebrew-calendar reflection experience built for the Common Era vibe coding sprint. Visitors move through a cinematic welcome, optionally enter their name and birthday, receive a symbolic personal thread, and continue into the current Hebrew month reading.

## What works

- Four-part animated welcome sequence
- Optional name and date-only birthday flow
- Deterministic Hebrew-date conversion and personal reading
- Twelve authored Hebrew-month profiles, glyphs, and constellation assets
- Current-month ritual and grounding note
- Responsive locked-mobile composition with reduced-motion support

## Still rough

- The final Friday/moon return panel is still a placeholder
- The interactive zodiac rotor is being developed separately
- Moon state is symbolic rather than astronomically exact
- Glyph and derived-asset licensing needs review before production distribution

## Built with

React, TypeScript, Vite, `@hebcal/core`, Codex, and GSD planning workflows.

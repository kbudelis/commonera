# Sketch Manifest

## Design Direction

A mobile-first celestial atlas: the immersive scroll/drag behavior of Gaspard Fery's Curve Gallery translated into the Cosmic Calendar world. The field uses the antique chart reference's deep ink blue and bone-white linework, with restrained amber reserved for focus and ritual moments. Constellations are original transparent SVG line drawings rather than framed photos, so the gallery feels suspended in a night sky. The experience should remain mystical, still, intelligent, and tactile rather than noisy or game-like.

The primary consultation flow now explores three structures against one fixed Tammuz content set: an editorial almanac, a functional lunar instrument, and a share-first ritual deck. This keeps the product's hidden Jewish-calendar system discoverable rather than front-loaded while testing how quickly a first-time visitor can move from a universal question to a useful reading, a personal profile, and a reason to return.

## Reference Points

- [Curve Gallery demo](https://tympanus.net/Tutorials/CurveGallery/) — camera travel along a curve, focus scaling, scroll/touch input.
- [gaspoorf/curve-gallery](https://github.com/gaspoorf/curve-gallery) — MIT-licensed source mechanics; no source photo assets reused.
- User-supplied antique celestial-chart image — palette and engraved linework direction.
- User-supplied Hiclipart constellation reference — transparency and isolated-figure direction only; original SVG artwork used here.
- Cosmic Calendar PRD §8 — deep indigo/night-sky base, warm gold/amber accent, functional celestial motifs.

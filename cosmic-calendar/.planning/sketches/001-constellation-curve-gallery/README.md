# Sketch 001: Constellation Curve Gallery

## Design Question

Does the Curve Gallery interaction translate into a useful, legible way to browse the Cosmic Calendar's twelve months on a phone?

## How to View

```bash
open .planning/sketches/001-constellation-curve-gallery/index.html
```

The sketch is optimized for a 390 × 844 portrait viewport. Swipe vertically or horizontally, use the wheel/trackpad, arrow keys, or enable Auto.

## Variants

- **A: Celestial Ribbon** — closest to the source: an irregular closed curve, strong focus scaling, and sparse transparent constellations.
- **B: Orrery** — a more ordered orbital path with amber focus marks and stronger month metadata.
- **C: Engraved Meridian** — a denser antique-chart treatment with coordinate lines, restrained scale, and a quieter archival feel.

## Selected Direction

**Variant A: Celestial Ribbon** is the provisional winner because it stays closest to the supplied Curve Gallery reference while giving the transparent constellation artwork enough scale and negative space on a phone.

## What to Look For

- Whether swiping feels natural enough to become the mobile browse interaction.
- Whether focused constellation art remains legible without opaque image cards.
- Whether the antique chart styling feels premium and still, rather than historical-theme decoration.
- Whether the current month information is discoverable without flattening the immersive field.

## Source Note

Interaction structure is independently reimplemented from the MIT-licensed Curve Gallery reference. The gallery photo assets and supplied third-party constellation image are not included. All constellation sprites in this sketch are original inline SVG line drawings with transparent backgrounds.

# Seder plate object illustrations

These eight original lithography-style illustrations were generated for the Let My People Host host guide on July 11, 2026. They depict the six standard seder-plate elements plus the optional orange and pomegranate additions supported by the intake.

Files:

- `zeroa.png` — roasted shank bone
- `beitzah.png` — roasted egg
- `maror.png` — horseradish root and prepared horseradish
- `charoset.png` — apple-and-walnut charoset
- `karpas.png` — parsley
- `chazeret.png` — romaine lettuce
- `orange.png` — orange addition
- `pomegranate.png` — pomegranate addition

The images were generated without reference images, text, logos, national symbols, or copyrighted characters. The shared art direction was a late-19th-century stone lithograph on a warm cream ground, using espresso linework with restrained cobalt, ochre, and brick-red accents. The source PNGs are 720×720 pixels and are intended for screen, print, and grayscale reproduction.
